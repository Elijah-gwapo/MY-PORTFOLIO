# Professional Portfolio Website

Welcome to my professional portfolio website! This is a modern, responsive portfolio featuring smooth animations and a clean design.

## Features

- Responsive design that works on all devices
- Smooth animations and transitions
- Interactive navigation
- Project showcase section
- Contact form
- Modern UI with gradient accents
- Mobile-friendly hamburger menu

## Technologies Used

- HTML5
- CSS3 (with Flexbox and Grid)
- JavaScript (with ES6+ features)
- Font Awesome for icons
- Google Fonts for typography

## Sections

- **Hero Section**: Introduction with call-to-action buttons
- **About Section**: Personal information and skills
- **Projects Section**: Showcase of projects with details
- **Contact Section**: Contact form and social links

## How to Run

To run this portfolio locally:

1. Clone or download this repository
2. Open the `index.html` file in your web browser
